TYPES OF CSS

1. Inline CSS

Applied directly to an element using the style attribute. Affects only that specific element. 
This is inline CSS.

2. Internal CSS (Embedded CSS)

Written inside the style tag within the head section of the HTML file. Affects the whole page but only that HTML document.

This is internal CSS.
